MANITKart

MANITKart is a dynamic setup to bring a solution to problems like selling and buying used products among college itself. It is a platform for Manitians where they can easily connect to buy or sell their products. This app is created on Android Studio using various tools and technologies like Firebase, Google APIs. Any authenticated user can post multiple number of ads and can even directly connect to the seller and buy the product.

Eat or Beat

This project aims to provide suggestions on food items based on previous dataset obtained through a fitness device like fitbit, using an android application.In App we have two options to send query to server, first is scanning the barcode of the product and then using speech recognition, for which we use Google Speech Recognition API. The output displays whether the food item is suitable or not along with a suggestion for better results.
